## 1) Architecture Assessment (decision-grade)
**For:** CIOs, program directors, enterprise architects  
**Outcome:** clarity on options, risks, and recommended direction  
**Typical outputs:** scope & constraints, option comparison, recommendation, risk register, next-step roadmap

## 2) Architecture Deliverables for Programs
**For:** programs needing strong governance alignment  
**Typical outputs:** Project Start Architecture, solution context, principles, key decisions, security & compliance alignment, stakeholder sign-off package

## 3) Rationalisation & Roadmaps
**For:** tool/platform consolidation and modernization  
**Typical outputs:** inventory model, classification, target state, migration strategy, prioritised roadmap, decision log

## Engagement model
1. Short intake (goals + constraints)  
2. Working sessions (stakeholders + evidence)  
3. Decision-ready deliverable + alignment
